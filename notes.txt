Verifique se o Docker está instalado e em execução:
Certifique-se de que o daemon do Docker está em execução. Você pode verificar isso com o comando:
## bash - sudo systemctl status docker

##  Se o Docker não estiver em execução, inicie-o com: bash - sudo systemctl start docker

Verifique a versão do Docker e do Docker Compose:
Verifique se você está usando versões compatíveis do Docker e do Docker Compose:
## bash - docker --version docker-compose --version


## Reinicie o Docker: bash - sudo systemctl restart docker

## Montar a imagem e subila: docker compose up
